# Calendar Level-Up Challenge

This project is a web application built using **TypeScript**, **Vite**, and **Motion (JavaScript animation library)**.

The application includes features such as a calendar system, daily challenges, weather information, animations, and sound effects.

---

## Technologies Used

- **TypeScript**
- **Vite**
- **Motion (JavaScript animation library)**
- **HTML / CSS**

## How to Run the Project

### 1. Install Dependencies

First install all project dependencies:

```bash
pnpm i
```

or

```bash
npm install
```

### 2. Start the Development Server

Run one of the following commands:

```bash
pnpm run dev
```

or

```bash
npm run dev
```

After running the command, open the local development URL shown in the terminal.
